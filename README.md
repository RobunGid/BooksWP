## Features

-   Interactive cart
-   Simple backend using FireBase
-   Form validation
-   Order sending to FireBase DB
-   Fetch book list from FireBase DB

## Tech Stack

**Client:** React, CSS Modules, Vite, ESlint, Prettier

**Server:** FireBase Realtime DB
